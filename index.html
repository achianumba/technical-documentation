Introduction

Being able to place elements where you want them on a webpage gives you better control over the layout of every site you build. While HTML elements are usually aligned to the left by default, there are various ways to vertically and/or horizontally position items on your webpage to fit the design you have in mind.

Traditionally, web-designers used the HTML <center> element to center HTML elements horizontally within containers. However, the <center> tag is now deprecated.

Enter CSS!

Cascading Style Sheets (CSS) offers multiple properties for centering elements (within their containers) on web pages. What makes these CCS properties better than the outdated HTML <center> element is that the <center> element only allows developers to align items horizontally while CSS properties vertically and/or horizontally center html elements depending on the developer's/client's preferences/needs.

Some CSS properties used for aligning elements on a web page include:

  • Text-Align
  • Vertical-align
  • Padding
  • CSS Grid
  • CSS Flexbox
Text-Align

The text-align property centers text content horizontally while the vertical-align property centers text content vertically within its container. 

                    
   /*Aligns the h2 element to the center (horizontally and vertically)*/
    h2 {
        text-align: center;
        vertical-align: middle;
        }

Note: The text-align and vertical-align properties also work on icons because web pages treat them as text.

Padding

Setting the same value for all sides (top, right, bottom, left) of the padding property of an element centers it horizontally and vertically within its container.

The sample CSS code below will center h2 elements horizontally and vertically.

                        
    h2 {
        padding: 12px;
        }
          OR

    h2 {
        padding: 12px 12px 12px 12px;
        }
Margin

You can easily center a block level element horizontally by setting the value auto to the CSS margin property.

Ensure you set the display property to block if it is an inline element you wish to center. For example, the code below horizontally positions all elements with the class "center" to the center of their containers. 

                    
     .center {
              width: 70%;
              margin: auto;
              }

                OR

             /*For inline elements*/
     .center {
              display: block;
              width: 70%;
              margin: auto;
              }

Note: The value of the width property of the element being centered must be less than the width of its parent element for this method to work.

CSS Grid

Set the CSS display property values to grid and then set the justify-content property to center to horizontally center child elements of the grid container.

Here is an example:

                    
     div {
          display: grid;
          justify-content: center;
          }

To vertically center the elements within the grid container, set the value of the align-items property to center as follows:

                    
     div {
          display: grid;
          align-items: center;
          }
CSS Flexbox

For the most part, CSS Flexbox has the same set of properties and values as CSS grid for centering HTML elements vertically or horizontally. The only major exception is that you have to set the display property value to flex as follows:

                    
      /* center align horizontally */
       div {
            display: flex;
            justify-content: center;
            }

      /* center align vertically */
       div {
            display: flex;
            align-items: center;
            }
